# LadyBugDodgeGame
Android game app - LadyBugDodge 무당벌레닷지
2인 개발 모바일 게임 프로젝트

**[게임명] : 무당벌레닷지**

**[게임 장르] : 모바일 캐주얼 게임**

**[제작 기간] : 2020/08/02 ~ 2021/02/14**
 
**[제작엔진/언어] : Unity / C#**

**[역할] : 기획 및 메인 프로그래밍**

**[게임 설명] :**
 - 애벌레를 먹고 속성(불/물/흙/전기/나무)을 얻는다.
 - 구슬을 먹어 무당벌레의 레벨을 올린다. 
 - 레벨을 올려 속성 공격을 강화한다.
 - 몬스터의 속성 총알 공격을 피한다.
 - 몬스터의 속성의 상성을 이용하여 공격한다.
